# Proyecto Final – Aplicación Web Interactiva

**Autor:** Denis Ariel Ramos  
**Materia:** Programación Web  
**Descripción:**  
Aplicación web desarrollada con HTML, CSS y JavaScript puro.  
Incluye login simulado, envío de correos reales mediante EmailJS,  
interacción con APIs públicas (imágenes, clima) y módulos dinámicos de operaciones y formularios.

**Tecnologías utilizadas:**
- HTML5
- CSS3
- JavaScript
- EmailJS API
- Open-Meteo API
- Random Dog / Picsum API
